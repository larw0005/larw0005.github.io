LIB2024- QA - Lindsay Larwill

  1. Question: What are the three components of Web?

    Answer: HTML, CSS, JavaScript

  2. Question: Define version control and give one example.

    Answer: It's a system that records changes to a file or set of files over time so that you can find and bring up specific versions later. An example of version control software is GitHub.

  3. Question: What tag is used to create the title of a web page? Give an example.

    Answer: <title>, example: <title>Week 1 Intro to HTML</title>.

  4. Question: What tag is used to create a paragraph in html? Give an example.

    Answer: <p>, example: <p>This is my paragraph</p>.

  5. Question: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason.

    Answer: The <p> element is a block-level element, while the others are all inline elements. Block elements stretch the full width availalbe, always start on a new line and has a top and bottom margin.

  6. Question: What are the tags required to create a navigation list in HTML?

    Answer: <nav>, <ul>, <li>

  7. Question: What element is used to create images? Name its attributes.

    Answer: <image> is used to create images, and its two attributes are src and alt. src is used to direct where to find the image and alt is used to describe the image.

  8. Question: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Answer: The <b> is the odd tag in this group. It is used for formatting text and making it bold. The other tags are used to define a table, rows in the table, and data/cells of the table in HTML.

  9. Question: Give an example of color picker input.

    Answer: <input type="color" id="firstcolor" name="firstcolor" value="#8B0000">

  10. Question: What is the use of name and value attributes? Give two examples.

    Answer: The name attribute specifies the name of an HTML element.
    The value attribute specifies the value of the element with thich it is used.
    This is an example of an HTML form with a name attribute: First Name: <input type="text" name="fname">
    This is an example of a value attribute added to a <button> element, which specifies the text on the button: <input type="button" value="Click here">

  11. Question: Give an example of inline style.

    Answer: <h2 style="color:red;text-align:center;">This is the style of heading #2</h2>

  12. Question: How would you link an external style sheet? Give example.

    Answer: The external style sheet would be linked inside the head section of your HTML document using the <link> element.
    Example: <link rel="stylesheet" type="text/css" href="css/style.css">

  13. Question: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Answer: div.absolute {
    position: absolute;
    left: 50%;
    top: 50%;
    width: 200px;
    height: 200px;
    margin: -100px 0 0 -100px;
    }

  14. Question: Create CSS keyframes block called "spin" that rotates an element.

    Answer:
    @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }
    }

  15. Question: Create CSS animation block to rotate an element that has the following properties :

    • class "rotate-me"
    • It rotates 5 times at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    Answer:
    @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }
    .rotate-me {
    animation-name: spin;
    animation-duration: 0.5s;
    animation-iteration-count: 5;
    animation-timing-function: linear;
    }

  16. Question: How would you include create an alert with string "I love JavaScript" using JavaScript?

    Answer:
    <script type="text/javascript">
    function showalert() {
    alert("I love JavaScript");
    }
    </script>

  17. Question: How would you fade out an element with id "fade-me" after a delay of 300ms using JQuery?

    Answer: $("fade-me").fadeOut(3000);

  18. Question: How would you select all elements with class "my-class" using Javascript and Jquery?

    Answer: JavaScript: var x = document.getElementsByClassName("my-class");
    JQuery: var x = document.querySelectorAll("my-class");

  19. Question: Give an example of input field that takes phone number with format "123-456-7890"?

    Answer: <label for="phone">Phone number:</label>
    <input type="tel" id="phone" name="phone" placeholder="123-456-7890">

  20. Question: Among HTML, CSS, Javascript, JQuery, what language did you like the best? State the reason.

    Answer: I liked HTML the best because it's the structural foundation of a webpage and there are certain tags, or root tags, that you need to have in your coding to build the page. I find it an easier language to work with because you always need to organize the page with the basic tags such as <header>, <body> and <html>. I understand that you can get very creative with styling in CSS and add more with Javascript but I like to keep it as simple as possible.
